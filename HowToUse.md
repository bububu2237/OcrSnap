# OcrSnap 操作手冊

Windows 截圖與標註工具，免安裝，解壓縮即可使用。

**系統需求：** Windows 10 1809（Build 17763）以上

---

## 快速開始

執行 `OcrSnap.exe`，程式會在右下角系統匣出現圖示，按下 **F4** 開始截圖。

---

## 截圖

| 操作 | 說明 |
|------|------|
| **F4** | 開始截圖（可在設定中更改） |
| 拖曳滑鼠 | 框選截圖範圍 |
| 滾輪 | 放大 / 縮小放大鏡 |
| **Enter** 或雙擊 | 確認截圖，進入標註視窗 |
| **Esc** | 取消截圖 |

---

## 標註工具

截圖確認後會開啟標註視窗，工具列由左至右：

| 圖示 | 工具 | 說明 |
|------|------|------|
| → | 箭頭 | 繪製箭頭 |
| ✏ | 鋼筆 | 自由筆跡 |
| □ | 矩形 | 繪製矩形框 |
| T | 文字 | 點擊後輸入文字，Enter 換行，Ctrl+Enter 確認 |
| ⬛模糊 | 高斯模糊 | 拖曳選取範圍進行模糊遮蔽 |
| 🎨 | 顏色 | 點擊變更筆刷顏色 |
| 粗細下拉 | 線條粗細 | 選擇 1 / 2 / 4 / 6 px |
| 複製圖示 | 複製 | 複製標註後的圖片到剪貼簿 |
| 釘選 | 釘選 | 將圖片釘選在螢幕上（浮動視窗） |
| OCR | 文字辨識 | 對截圖進行 OCR 文字辨識 |
| Ctrl+Z | 復原 | 復原上一步標註 |

---

## OCR 文字辨識

點擊工具列 **OCR** 按鈕，或對釘選圖片右鍵選擇「OCR 辨識」。

### 辨識結果視窗

| 功能 | 說明 |
|------|------|
| 全選並複製 | 複製所有辨識文字 |
| 辨識框 | 開啟 / 關閉文字區域框線顯示 |
| 語言選單 | 切換辨識語言（繁體中文、簡體中文、English、日本語、한국어） |
| 重新辨識 | 以選定語言重新辨識整張圖片 |
| **手動框選辨識** | 在右側圖片預覽上拖曳選取局部區域，放開後自動辨識該區域 |

> **注意：** OCR 使用 Windows 內建辨識引擎，需要對應語言包已安裝。
> 安裝方式：設定 → 時間與語言 → 語言與地區 → 新增語言，下載語言包後即可使用。

---

## 釘選視窗

截圖後按釘選按鈕，或直接在截圖框選後按 **P** 鍵，可將截圖釘在螢幕上。

| 操作 | 說明 |
|------|------|
| 拖曳 | 移動釘選視窗 |
| 滾輪 | 縮放圖片大小 |
| 右鍵選單 | 複製、OCR 辨識、關閉 |
| **Esc** 或關閉按鈕 | 關閉釘選視窗 |

---

## 顏色吸取

截圖視窗放大鏡中央顯示目前游標下的顏色，確認截圖前可按 **C** 複製顏色值到剪貼簿。

---

## 設定

右鍵點擊系統匣圖示 → **設定**，可更改截圖熱鍵組合。

設定檔儲存於：`%AppData%\OcrSnap\settings.json`

---

## 結束程式

右鍵點擊系統匣圖示 → **結束**。
